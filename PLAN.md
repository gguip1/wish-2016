# 2026 첫 소원 보드 - 프로젝트 기획서

## 프로젝트 개요

**목표**: 2026년 새해를 맞아 사람들이 첫 소원을 공유하는 실시간 보드
**기한**: 1시간 내 개발 + 배포
**유효 기간**: 2025년 12월 31일 ~ 2026년 1월 1일 (1월 1일까지만 의미있는 프로젝트)

---

## 핵심 기능

### 1. 카운트다운 타이머
- 2026년 1월 1일 자정까지 실시간 카운트다운
- 일/시/분/초 표시
- 자정이 되면 축하 애니메이션

### 2. 소원 작성
- 익명으로 소원 작성 (100자 제한)
- 작성 즉시 목록에 반영

### 3. 실시간 소원 목록
- 최신순 정렬
- 5초마다 자동 갱신 (polling)
- 새 소원 등장 시 애니메이션

---

## 인터랙티브 요소 (중요!)

### 애니메이션 & 효과
1. **배경**: 밤하늘 + 반짝이는 별 파티클
2. **카운트다운**: 숫자 flip 애니메이션
3. **소원 카드**:
   - 등장 시 fade-in + slide-up
   - hover 시 살짝 확대 + glow 효과
4. **소원 제출**:
   - 버튼 클릭 시 ripple 효과
   - 성공 시 confetti 폭죽
5. **자정 도달 시**: 대형 축하 애니메이션 + 폭죽

### 마이크로 인터랙션
- 입력 필드 focus 시 glow
- 글자 수 카운터 (100자 접근 시 색상 변화)
- 스크롤 시 소원 카드 stagger 애니메이션

## 기술 스택

| 구분 | 기술 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 14 (App Router) | 풀스택, 빠른 개발 |
| 스타일링 | Tailwind CSS | 빠른 스타일링 |
| 애니메이션 | Framer Motion | 부드러운 인터랙션 |
| 파티클 | canvas-confetti | 폭죽 효과 |
| 데이터베이스 | SQLite (better-sqlite3) | 심플, 파일 하나 |
| 배포 | Railway | SQLite 호환 |

## 데이터베이스 스키마

```sql
CREATE TABLE wishes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT NOT NULL,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_wishes_created_at ON wishes(created_at DESC);
```

## API 설계

### GET /api/wishes
최신 소원 50개 조회

**Response**:
```json
{
  "wishes": [
    {
      "id": 1,
      "content": "건강하게 지내고 싶어요",
      "created_at": "2025-12-31T23:59:00.000Z"
    }
  ]
}
```

### POST /api/wishes
새 소원 작성

**Request**:
```json
{
  "content": "2026년에는 취업 성공!"
}
```

**Response**:
```json
{
  "success": true,
  "wish": {
    "id": 2,
    "content": "2026년에는 취업 성공!",
    "created_at": "2025-12-31T23:59:30.000Z"
  }
}
```

**유효성 검사**:
- content: 1~100자
- 빈 문자열/공백만 있는 경우 거부
- HTML 태그 이스케이프 (XSS 방지)
